Política de Privacidad

Última actualización: 01 de diciembre de 2025

Esta Política de Privacidad describe cómo la aplicación FocusTimer, desarrollada por Exvacorp, recopila, utiliza y protege la información proporcionada por los usuarios. El uso de la aplicación implica la aceptación de esta política.

1. Información que recopilamos

FocusTimer utiliza servicios de Firebase, los cuales pueden recopilar información técnica y estadística de manera anónima. La aplicación no recopila información personal identificable.

1.1 Firebase Analytics

Utilizamos Firebase Analytics para comprender cómo los usuarios interactúan con la aplicación. Se recopila información como:

  • Eventos dentro de la app (sesiones, clics, navegación)
  • Modelo del dispositivo
  • Versión del sistema operativo
  • Información técnica y de rendimiento

Estos datos son anónimos y no permiten identificar al usuario.

1.2 Firebase Crashlytics

Recopilamos datos relacionados con fallos o cierres inesperados de la aplicación. Estos datos incluyen:

  • Modelo del dispositivo
  • Versión de Android
  • Estado técnico al momento del error
  • Registros del fallo

La información se utiliza exclusivamente para mejorar la estabilidad y rendimiento del servicio.

1.3 Firebase Cloud Messaging

FocusTimer utiliza Firebase Cloud Messaging (FCM) para enviar notificaciones relacionadas con el funcionamiento de la app. Firebase puede recopilar:

  • Identificador del dispositivo para la entrega de notificaciones
  • Datos técnicos necesarios para su funcionamiento

No se recopilan datos personales.

2. Uso de la información recopilada

Los datos recopilados se utilizan para:

  • Mejorar la experiencia del usuario
  • Optimizar el rendimiento de la aplicación
  • Analizar estadísticas de uso anónimo
  • Detectar y corregir errores
  • Enviar notificaciones relevantes

3. Compartición de datos

No vendemos ni compartimos datos con terceros, excepto con proveedores necesarios para el funcionamiento de la aplicación, como Google/Firebase. Todos los datos se manejan bajo las políticas de privacidad de Google.

4. Seguridad de la información

Firebase implementa medidas de seguridad estándar para proteger la información durante su transmisión y almacenamiento. Aunque ningún sistema es 100% infalible, trabajamos para mantener la seguridad adecuada.

5. Derechos del usuario

Dependiendo de su región, el usuario puede:

  • Solicitar la eliminación de datos anónimos recopilados por Firebase
  • Desactivar el envío de datos desde configuraciones del dispositivo
  • Desinstalar la aplicación para detener toda recopilación de información

6. Servicios externos

Para más información sobre cómo Firebase gestiona los datos, puede consultar:

  • Política de Privacidad de Google
  • Política de Privacidad de Firebase

7. Cambios en esta política

Podemos actualizar esta Política de Privacidad en cualquier momento. Cualquier modificación será publicada en esta misma página.
