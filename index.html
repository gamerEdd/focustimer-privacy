Política de Privacidad

Última actualización: 01 de diciembre de 2025

Esta Política de Privacidad explica cómo la aplicación FocusTimer, desarrollada por Exvacorp, recopila y utiliza información del usuario.

1. Información que recopilamos

FocusTimer utiliza servicios de Firebase que pueden recopilar información técnica y de uso anónima.

1.1 Firebase Analytics

Recopila datos anónimos para entender cómo se usa la aplicación:

  • Eventos dentro de la app (sesiones, tiempos, clics, navegación)
  • Modelo del dispositivo
  • Versión del sistema operativo
  • Información general del dispositivo

No se recopilan datos que identifiquen al usuario.

1.2 Firebase Crashlytics

Recopila información cuando ocurre un fallo en la app:

  • Modelo del dispositivo
  • Versión de Android
  • Estado interno al momento del fallo
  • Datos técnicos del error

Estos datos se usan para mejorar la estabilidad de la app.

1.3 Firebase Cloud Messaging (Notificaciones)

Se utiliza para enviar notificaciones relevantes. Firebase puede recopilar:

  • Identificador del dispositivo para recibir notificaciones
  • Datos técnicos necesarios para la entrega del mensaje

No se recopila información personal identificable.

2. Para qué usamos los datos

  • Mejorar la experiencia y rendimiento de la aplicación
  • Entender el uso de la app mediante estadísticas anónimas
  • Detectar y solucionar errores
  • Enviar notificaciones útiles relacionadas al funcionamiento

3. Compartición de datos

Los datos no se venden ni se comparten con terceros excepto con servicios necesarios:

  • Google Firebase (Analytics, Crashlytics, Messaging)

4. Seguridad

Firebase utiliza medidas estándar de seguridad para proteger los datos en tránsito y almacenamiento.

5. Derechos del usuario

Dependiendo de la región, los usuarios pueden:

  • Solicitar eliminación de datos anónimos
  • Deshabilitar la recopilación desde ajustes del dispositivo
  • Dejar de usar la app y desinstalarla

6. Servicios externos

Recomendamos revisar las políticas de los servicios utilizados:

  • Política de Google / Firebase

7. Modificaciones

Podemos actualizar esta Política cuando sea necesario. Indicaremos cambios en esta misma página.
