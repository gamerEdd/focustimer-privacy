CoreFlow - Política de Privacidad y Eliminación de Cuenta

Aplicación: CoreFlow

Desarrollador: exvacorp

Última actualización: 2026

1. Información que recopilamos

CoreFlow puede recopilar información como nombre, correo electrónico, identificadores de dispositivo y datos de uso de la aplicación con el fin de mejorar la experiencia del usuario.

2. Uso de la información

La información recopilada se utiliza para:

  • Proveer y mejorar el servicio
  • Personalizar la experiencia del usuario
  • Enviar notificaciones importantes

3. Compartición de datos

No vendemos datos personales. Algunos datos pueden ser compartidos con servicios externos (como Firebase o APIs) necesarios para el funcionamiento de la aplicación.

4. Seguridad

Los datos se transmiten mediante protocolos seguros (HTTPS) para proteger la información del usuario.

5. Eliminación de cuenta (Account Deletion)

Si has creado una cuenta en la aplicación CoreFlow, tienes derecho a solicitar la eliminación de tu cuenta y de todos los datos asociados.

Cómo solicitar la eliminación

Para eliminar tu cuenta, envía un correo electrónico a:

eddgtx1@gmail.com

Con el asunto: "Eliminar cuenta CoreFlow"

Incluye información suficiente para identificar tu cuenta (por ejemplo, correo registrado).

Datos que se eliminarán

  • Cuenta de usuario
  • Datos personales asociados
  • Información almacenada dentro de la aplicación

Tiempo de procesamiento

Las solicitudes serán procesadas en un plazo máximo de 7 días.

Retención de datos

Algunos datos pueden conservarse temporalmente si es necesario para cumplir con obligaciones legales o de seguridad.

6. Cambios en esta política

Podemos actualizar esta política en cualquier momento. Se recomienda revisarla periódicamente.

7. Contacto
